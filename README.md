# Anotações e Estudos de TI

Repositório pessoal com anotações, trilhas e materiais de estudo sobre desenvolvimento de software e fundamentos de TI.

O objetivo é manter um lugar organizado para revisar conceitos, acompanhar evolução dos estudos e guardar referências úteis.

## Áreas de estudo

- **backend/**: estudos de PHP, arquitetura, redes, grafos e materiais de apoio para back-end.
- **git/**: conceitos iniciais, branches, controle de versão e padrões de commits.
- **testes/**: tipos de testes e abordagens de testes de software.
- **sistemas-operacionais/**: arquitetura de computadores, Unix, processos, memória e entrada/saída.
- **system-design/**: introdução, roadmap e fundamentos de arquitetura de sistemas.
- **design-patterns/**: anotações sobre padrões de projeto.
- **metodologias-ageis/**: Scrum, Kanban e Scrumban.
- **recursos/**: links e referências úteis.

## Trilhas

- **trilhas/backend/**: checklists, cronogramas e plano de estudo para evolução em back-end.

## Projetos

- **projetos/**: documentação e materiais de projetos pessoais, como arquitetura de APIs e canvas de planejamento.

## Atualização de conteúdo

O conteúdo é atualizado conforme avanço nos estudos. Algumas anotações podem estar incompletas, em revisão ou servindo como rascunho de aprendizado.

## Contribuindo

Este é um repositório pessoal de estudos, mas sugestões, correções e melhorias são bem-vindas por issue ou pull request.
